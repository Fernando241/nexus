# NEXUS

Professional SaaS-oriented backend platform built with Laravel and React.

## Goals

- Build a scalable modular monolith architecture
- Apply clean backend engineering practices
- Develop production-oriented API architecture
- Integrate authentication, authorization, queues, caching, and async processing
- Simulate real-world SaaS engineering workflows

## Stack

- Laravel
- PHP 8.4
- Docker
- MySQL
- Redis
- React
- Vite

## Architecture

NEXUS follows a pragmatic modular monolith architecture organized by business domains.

## Environment

Development environment is fully containerized using Docker Compose.

## Status

Foundation phase in progress.
